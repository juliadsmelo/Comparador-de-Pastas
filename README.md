# Comparador-de-Pastas
**Comparação de Estruturas de Pastas**

Este script realiza a comparação entre duas estruturas de pastas (antiga e nova), identificando arquivos e subpastas que estão presentes na pasta de referência (antiga) mas ausentes na pasta de destino (nova). Pastas vazias são automaticamente desconsideradas da análise. O resultado é exibido de forma agrupada no terminal — organizado por subpasta — e exportado para um arquivo Excel (.xlsx) com filtros automáticos, permitindo uma análise detalhada dos itens faltantes. Compatível com pastas sincronizadas via OneDrive/SharePoint (Arquivos Sob Demanda), sem necessidade de download do conteúdo.
